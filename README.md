## Setup
- Get TMDb API key
- Add key in Constants.swift
- iOS minimum 16.6
- Support iPhone and iPad potrait orientation only.

## Architecture
- MVVM
- Clean separation of concerns

## Features
- Popular Movies
- Search
- Movie Details
- Youtube Trailer Playback
- Favorites Using UserDefaults

## Assumptions
- Internet required
- YouTube trailers available

## Known Limitations
- No offline caching
- Limited error handling
